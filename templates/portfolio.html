About Hamza!

Hello everybody. I'm Hamza Boughraira from Tunisia, I'm a Python/Django and IoT developer, a junior Linux administrator and a book writer.

My career as a developer started four years ago, I wasn't a software developer at first, I was a hardware and embedded systems programmer and I've created lots of stuff, including smart robots and so on. In the last two years, I focused on software development and more precisely on back-end web development, so I taught myself the Python programming language and the Django web framework, as well as relational database design and SQL, JavaScript and HTML, Node.js and Express.js and of course the Linux system administration.
